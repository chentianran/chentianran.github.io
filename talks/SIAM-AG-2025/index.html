How many zeros?

How many zero does \[ a x^2 + b x + c \] have?

Interpretation:

  • Counting complex zeros
  • Counting finite zeros only
  • Counting with multiplicity

Answer:

  • 2 (generic answer)
  • 1 (if $a = 0, b \ne 0$)
  • 0 (if $a,b = 0, c \ne 0$)

How many zero does \[ \left\{ \begin{aligned} &a_1 x^2 + a_2 y^2 + a_3 x^2 y^2 + a_4 \\ &b_1 x^2 y + b_2 x y^2 + b_3 \end{aligned} \right. \] have?

General setup

Consider a Laurent system $F$ with supports $A_1,\ldots,A_n$ span $\mathbb{R}^n$ (otherwise no isolated zeros). For generic coefficients, zeros of $F$ are nonsingular and the total number is a constant.

(Bernshtein's 1st Theorem) This constant is the mixed volume $\text{MV}(A_1,\ldots,A_n)$ (BKK bound).

Bernshtein, The number of roots of a system of equations. Funct Anal Appl+ 9, 183-185 (1975).

For specific coefficients, number of isolated zeros of $F$ to be strictly less than the BKK bound (even counting multiplicity).

  • Zeros fly off to toric infinity;
  • Isolated zeros absorbed into positive dimensional components;
(Bernshtein's 2nd) These can happen only when there exists a nonzero vector $\mathbf{v}$ such that $\operatorname{init}_{\mathbf{v}} F$ has nontrivial zeros.

At most 6 zeros

$E$ is 1-dimensional and is defined by a binomial system.

It contains a "torus orbit": \[ (k_{10},k_{12},k_{32},k_{30},w_1,w_2,w_3)(t) = (a t^1, a t^1, a t^1, a t^1, b t^0, c t^0, d t^0) \] is in $E$ for all $t \in \mathbb{C}^*$. ($a,b,c,d \ne 0$ are arbitrary complex numbers)

(C., Lindberg, Li) N.o. zeros for generic $(\mathbf{k},\mathbf{w}) \in E$: \[ \sum_{v} \text{Root count of } \operatorname{init}_v F(\mathbf{x}; a t^1, a t^1, a t^1, a t^1, b t^0, c t^0, d t^0) = 6 \] for generic $a,b,c,d \ne 0$. (Strictly less than BKK bound: 12)

One of the initial system \[ \begin{aligned} \operatorname{init}_v f_1 &= k_{10} \class{highlight}{x_1} + k_{12} \class{highlight}{x_1} \class{highlight}{x_2}^{-1} - w_1 \\ \operatorname{init}_v f_2 &= -k_{21} \class{highlight}{x_1} \class{highlight}{x_2}^{-1} - k_{23} \class{highlight}{x_3} \class{highlight}{x_2}^{-1} - w_2 \\ \operatorname{init}_v f_3 &= k_{32} \class{highlight}{x_3} \class{highlight}{x_2}^{-1} + k_{30} \class{highlight}{x_3} - w_3 \end{aligned} \] normally has 2 solutions. But under the conditions

\[ \left\{ \begin{aligned} k_{10} k_{12} k_{32}^{-1} k_{30}^{-1} &= 1 \\ k_{10} k_{12}^{-1} k_{32}^{-1} k_{30} &= 1 \\ k_{10} k_{12}^{-1} k_{32} k_{30}^{-1} &= 1 \end{aligned} \right. \]

the initial system has only 1 solution.

In total, there are 6 similar initial systems each producing 1 solution. Therefore, the total zero count is \[ 1 + 1 + 1 + 1 + 1 + 1 = 6 \]

Torically discriminated system

Consider a square system of $n$ Laurent polynomials \[ F(\mathbf{x}; \mathbf{c}) = F(x_1,\ldots,x_n; c_1,\ldots,c_m) \] parametrized by its coefficients $c_1,\ldots,c_m$ (one for each monomial) with supports $A = (A_1,\ldots,A_n)$ spanning $\mathbb{R}^n$.

Coefficients do not provide a crisp parametrization. We must form the quotient modulo a group actions. Coefficients can still be proxy parameters.

For generic coefficients $\mathbf{c}$, the $\mathbb{C}^*$-root count is a positive constant.

(Bernshtein) This constant is $\operatorname{MV}(A_1,\ldots,A_n)$. (BKK)

We are interested in: There is a set of $E$ of parameters such that \[ \text{Isolated $\mathbb{C}^*$-zero count of } F(\mathbf{x}; \mathbf{c}) \lneq \text{BKK bound} \quad \forall \mathbf{c} \in E. \]

How many zeros? What happened to the missing ones?

Special case: Torus orbit in $E$

We are interested: There is a set of $E$ of parameters such that \[ \text{Isolated zero count of } F(\mathbf{x}; \mathbf{c}) \lneq \text{BKK bound} \quad \forall \mathbf{c} \in E. \]

A 1-parameter torus orbit in $E$ is the image of a map \[ t \;\mapsto\; \mathbf{c} \circ t^{\mathbf{w}} = (c_1 t^{w_1} , \ldots, c_m t^{w_m}) \in E \quad\text{for all } t \in \mathbb{C}^* \]

Not all such torus orbits are interesting.

An orbit $t \mapsto \mathbf{c} \circ t^{\mathbf{w}}$ in $E$ is trivial if $\mathbf{w}$ is in the row space of the Cayley embedding matrix $\hat{A}$. (...and consider other symmetries)
(C., Lindberg; C., Korchevskaia, Li) The $\mathbb{C}^*$-zero count for generic choices of coefficients in $E$ \[ \sum_{v} \text{$\mathbb{C}^*$-zero count of } \operatorname{init}_v F(\mathbf{x}; \mathbf{c} \circ t^{\mathbf{w}}) \] for generic $\mathbf{c} \in E$.
The root count for generic parameters in $E$ \[ \sum_{v} \text{Root count of } \operatorname{init}_v F(\mathbf{x}; \mathbf{c} \circ t^{\mathbf{w}}) \] for generic $\mathbf{c} \in E$.

Applying this to the Kuramoto equations

For the algebraic Kuramoto (PV power flow) system \[ \sum_{j=0}^{N-1} k_{ij} \left( \frac{x_i}{x_j} - \frac{x_j}{x_i} \right) \] derived from an even cycle $C_N$. Counting mulitplicity, the complex root count is \[ \left[ N \binom{N-1}{N/2 - 1} - \beta \right], \] where $\beta$ is the number of balanced cycles. (Less than BKK bound)

  • 2 nonsingular isolated zeros \[ \{ (1,1,1), (-1,1,-1) \}\]
  • 3 one-dimensional components parametrized by \[ \begin{aligned} &\left(-2 i \lambda, -1, 2 i \lambda\right), & &\left( 2 i \lambda, -1, \frac{1}{ 2 i \lambda} \right ), & &\left( 1 / 2 i \lambda, \frac{-1}{4\lambda^2}, \frac{-1}{2 i \lambda} \right), \end{aligned} \]

Stratified polyhedral homotopy method

There is "stratified" polyhedral homotopy $H(\mathbf{x},t)$ defining 12 smooth homotopy paths

  • 2 nonsingular isolated zeros
  • 3 one-dimensional components
  • 2 embedded points
  • 1 paths converging to each (2)
  • 2 paths converging to each (6)
  • 2 paths converging to each (4)

\[ \text{BKK bound} = 12 = \underbrace{1 + 1 }_{\substack{\text{Isolated}\\\text{points}}} + \underbrace{2 + 2 + 2}_{\substack{\text{1-dimensional}\\\text{components}}} + \underbrace{2 + 2 }_{\substack{\text{Embedded}\\\text{points}}} \]

Every homotopy path converge to a point in $(\mathbb{C}^*)^3$: (If counted "correctly") the intersection index in $(\mathbb{C}^*)^3$ equal the BKK bound.
