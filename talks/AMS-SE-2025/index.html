How electricity work

\[ W = \text{work} \]

\[ v(t) = \frac{dW}{dq} \] Voltage (abused notation)

\[ q = \text{charge} \]

\[ i(t) = \frac{dq}{dt} \] Current

\[ \frac{dW}{dt} = \frac{dW}{dq} \, \frac{dq}{dt} \]

\[ p(t) = v(t) \, i(t) \]

Instantaneous power equation

\[ p(t) = v(t) \, i(t) \]

Phasor representation of steady state AC solution \[ P + \mathfrak{i} Q = V \, \overline{I} \quad \text{where} \left\{ \begin{aligned} V &= V_{max} e^{\mathfrak{i} \theta_V} & P &: \text{ real power} \\ I &= I_{max} e^{\mathfrak{i} \theta_I} & Q &: \text{ reactive power} \end{aligned} \right. \]

Nodal analysis: Each bus has its own $P_i, Q_i, V_i, I_i$.

Nodal analysis [Image]. In Wikipedia, The Free Encyclopedia. Retrieved Sep 7, 2024, from https://en.wikipedia.org/wiki/Nodal_analysis

Kirchhoff's Current Law (KCL) $+$ Ohm's Law in AC Circuits:

\[ P_i + \mathfrak{i} Q_i = \sum_{j=0}^n \overline{Y}_{ij} V_i \overline{V_j} \quad\text{for each bus } i=1,\ldots,n \]

$Y_{ij}$: Nodal admittance.

Power flow equation

\[ P_i + \mathfrak{i} Q_i = \sum_{j=0}^n \overline{Y}_{ij} V_i \overline{V_j} \quad\text{for each bus } i=1,\ldots,n \]

Bus type Constants Unknowns
Slack bus $|V_i|,\arg(V_i)$ $P_i,Q_i$
PV bus $|V_i|,P_i$ $\arg(V_i),Q_i$
PQ bus $P_i,Q_i$ $|V_i|,\arg(V_i)$

A real power network always operates at a solution to this system of equations (operating point).

Learning about the landscape of all (or a large portion of) solutions is an important problem.

All PQ system

\[ P_i + \mathfrak{i} Q_i = \sum_{j=0}^n \overline{Y}_{ij} \class{highlight}{V_i} \class{highlight}{\overline{V_j}} \quad\text{for each bus } i=1,\ldots,n \]

Unknowns: $\class{highlight}{V_1},\ldots,\class{highlight}{V_n}$. $n$ equations, $n$ unknowns. Not algebraic.


Turning this into a polynomial system:
  • Use variables $x_i,y_i$ with $V_i = x_i + \mathfrak{i} y_i$
  • Use variables $r_i,c_i,s_i$ with $V_i = r_i (c_i + \mathfrak{i} s_i)$ and $c_i^2 + s_i^2 = 1$.

Different algebraic formulation create different numbers of extraneous complex solutions.

Wirtinger coordinates

\[ P_i + \mathfrak{i} Q_i = \sum_{j=0}^n \overline{Y}_{ij} \class{highlight}{V_i} \class{highlight}{\overline{V_j}} \quad\text{for each bus } i=1,\ldots,n \]

Let $\class{highlight}{U_i} = \class{highlight}{\overline{V}_i}$. Separating holomorphic and anti-holomorphic parts in the equations.

\[ P_i + \mathfrak{i} Q_i = \sum_{j=0}^n \overline{Y}_{ij} \class{highlight}{V_i} \class{highlight}{U_j} \quad\text{for each bus } i=1,\ldots,n \]

After relaxing $U_i = \overline{V}_i$, we have a polynomial system.


Problem: $2n$ unknowns, $n$ equations (not square)

\[ P_i + \mathfrak{i} Q_i = \sum_{j=0}^n \overline{Y}_{ij} \class{highlight}{V_i} \class{highlight}{U_j} \quad\text{for each bus } i=1,\ldots,n \]

\[ P_i - \mathfrak{i} Q_i = \sum_{j=0}^n Y_{ij} \class{highlight}{U_i} \class{highlight}{V_j} \quad\text{for each bus } i=1,\ldots,n \]

$2n$ complex unknowns, $2n$ equations

Bernshtein-generalness

\[ \left\{ \begin{aligned} P_i + \mathfrak{i} Q_i &= \sum_{j=0}^n \overline{Y}_{ij} \class{highlight}{V_i} \class{highlight}{U_j} \quad\text{for } i=1,\ldots,n \\ P_i - \mathfrak{i} Q_i &= \sum_{j=0}^n Y_{ij} \class{highlight}{U_i} \class{highlight}{V_j} \quad\text{for } i=1,\ldots,n \end{aligned} \right. \]

Bernshtein-general


Question. When is the Bernshtein-generalness broken?

Question. When is the Bernshtein-generalness broken?

Why do we care?

Computational evidence suggests that real power networks break Bernshtein-generalness.

Dan Wu, Bin Wang, July 1, 2019, "Collection of numerous power flow solutions of standard ieee test systems", IEEE Dataport, doi: https://dx.doi.org/10.21227/24bh-hj72

Good source of counterexamples to engineering assumptions.

There exist choices of parameters for which the power flow equation has infinitely many (real) solutions.

The $C_4$ example

Answer: "Balanced" cycle

The PQ power flow system on $C_4$

\[ \left\{ \begin{aligned} P_i + \mathfrak{i} Q_i &= \sum_{j=0}^n \overline{Y}_{ij} \class{highlight}{V_i} \class{highlight}{U_j} \quad\text{for } i=1,2,3 \\ P_i - \mathfrak{i} Q_i &= \sum_{j=0}^n Y_{ij} \class{highlight}{U_i} \class{highlight}{V_j} \quad\text{for } i=1,2,3 \end{aligned} \right. \]

has 16 isolated complex solutions for generic $Y_{ij} \in \mathbb{C}$, $P_i, Q_i \in \mathbb{R}$.

If nonzero $Y_{ij}$ satisfies the equation \[ \color{blue}{Y_{30} Y_{12}} = \color{red}{Y_{10} Y_{32}} \] then there are less than 16 complex isolated solutions.

General even cycles

For a cycle network $C_N$ with even $N$

\[ \left\{ \begin{aligned} P_i + \mathfrak{i} Q_i &= \sum_{j=0}^n \overline{Y}_{ij} \class{highlight}{V_i} \class{highlight}{U_j} \quad\text{for } i=1,\ldots,N-1 \\ P_i - \mathfrak{i} Q_i &= \sum_{j=0}^n Y_{ij} \class{highlight}{U_i} \class{highlight}{V_j} \quad\text{for } i=1,\ldots,N-1 \end{aligned} \right. \]

(Rob Davis & C. 2022) This system has \[ N 2^{N-2} \] isolated complex solutions for generic parameters.

Davis & Chen Computing Volumes of Adjacency Polytopes via Draconian Sequences Electron J Comb 29, (2022).

If there is a partition $E_+ \cup E_-$ of equal size and \[ \prod_{ij \in E_+} \color{blue}{Y_{ij}} = \class{highlight}{(-1)^{N/2}} \prod_{ij \in E_-} \color{red}{Y_{ij}} \] then there are less than $N 2^{N-2}$ isolated complex solutions.

Cycle basis description

(C., Korchevskaia, Lindberg) Let $G$ be a graph consisting of $d$ independent even cycles, $C_1,\ldots,C_d$ that overlap at a single edge. Let $U_i \subset \{ -1, 0, +1 \}^{|E(G)|}$, for $i=1,\ldots,d$ be the set of balanced primitive cycle vectors associated with $C_1,\ldots,C_d$. If
\[ \sum_{j=1}^d (-1)^{|C_{i_j}| / 2} Y^{\vec{\eta}_{i_j}} = 1 \text{ where } (\vec{\eta}_{i_1},\ldots,\vec{\eta}_{i_t}) \in U_{i_1} \times \cdots \times U_{i_t} \text{ is consistent} \]
then the system of algebraic power flow equations is not Bernshtein-general.

Kuramoto equations

The Kuramoto model is a simple dynamical system that models the nonlinear interaction among weakly coupled oscillators.

Kuramoto. 1975, Self-entrainment of a population of coupled non-linear oscillators doi:10.1007/bfb0013365.

\[ \overline{w} = w_i - \sum_{j \sim i} k_{ij} \sin(\theta_i - \theta_j) \]

  • Simple enough to be analyzed
  • Complex enough to exhibit interesting emergent behavior

Laurent polynomial forumulation \[ 0 = w_i - \sum_{j \in \mathcal{N}_G(i)} \frac{k_{ij}}{2 \mathfrak{i}} \left( \frac{x_i}{x_j} - \frac{x_j}{x_i} \right) \quad\text{for } i = 1,\ldots,n \] in $n$ complex unknowns $x_1,\ldots,x_n \ne 0$ ($x_0 = 1$ is a constant).

Constructing positive-dimensional solutions

(C., Korchevskaia, Lindberg) Let $G$ be a unicycle graph on $n+1$ nodes that contains a unique even cycle $O$. Suppose the coupling coefficients satisfy:
  • $(G,K)$ contains a balanced subnetwork $(\vec{H},K)$;
  • $|k_{ij}|$ are identical for edges on $O$.

Then algebraic Kuramoto system derived from the network of identical oscillators has a positive-dimensional $\mathbb{C}^*$-zero set.

Indeed we can construct a $\mathbb{C}^*$-orbit \begin{align*} \begin{bmatrix} x_1(\lambda) & \cdots & x_m(\lambda) \end{bmatrix} &= ( \lambda \cdot \boldsymbol{\eta}_{\vec{T}} \circ (\mathbf{k}(\vec{T})^{-I}) )^{Q(\vec{T})^{-1}} \\ x_i &= \pm x_{\pi(i)} \quad\text{for } i = m+1, \ldots, n, \end{align*}

(Require further explanation) Main point: It contain an 1-dimensional orbit parametrized by a system of Laurent monomials.

Real-orbits

...... Furthermore, if $k_{ij}$'s are all identical, then the $\mathbb{C}^*$-orbit constructed contains a $S^1$-robit in the real torus.

For the 4-cycle example, when the natural frequencies are identical, and $k_{ij}$'s are identical, we have 1-dimensional solution sets \[ \left\{ \begin{aligned} \theta_0 &= 0 \\ \theta_1 &= t + \pi + \sigma^{10} \\ \theta_2 &= \pi + \sigma^{12}_{10} \\ \theta_3 &= t + \sigma^{30} \\ \end{aligned} \right. \quad \left\{ \begin{aligned} \theta_0 &= 0 \\ \theta_1 &= \sigma^{10} + t \\ \theta_2 &= \pi + \sigma^{21}_{10} \\ \theta_3 &= \sigma^{30} - t \end{aligned} \right. \quad \left\{ \begin{aligned} \theta_0 &= 0 \\ \theta_1 &= \sigma^{10} - t \\ \theta_2 &= \sigma^{21}_{10} -2t \\ %\sigma(k_{21}/k_{10}) \\ \theta_3 &= \pi + \sigma^{30} -t \\ \end{aligned} \right. \] $\sigma$'s are phase shift values, $0$ or $\pi$, that only used to adjust for the signs of $k_{ij}$'s (not very important).

Beyond unicycle networks

Proposition. Suppose $G$ consists of $d$ independent even cycles $C_1,\ldots,C_d$ that share a single edge $e$, then with the choice of the coupling coefficients \[ k_{ij} = \begin{cases} sd &\text{if } \{i,j\} = e \text{ (shared edge)}\\ s &\text{othewise}, \end{cases} \] for any $s \in \mathbb{C}^*$, and identical natural frequencies, the algebraic Kuramoto system has a positive-dimensional $\mathbb{C}^*$-zero set.

There is a similar $\mathbb{C}^*$-orbit formula that parametrizes an 1-dimensional orbit inside the zero set.

...... there also can be real orbits in the solution set.
