Bernshtein-generalness, again

(C., Korchevskaia, Lindberg) For generic $P_i$'s and generic but symmetric $a_{ij},b_{ij}$, the number of complex solutions is \[ \operatorname{Vol}( \; P_G \; ) = \operatorname{Vol}( \operatorname{conv} \{ \pm (e_i - e_j) \mid (i,j) \in E(G) \} ) \] On the typical and atypical solutions to the Kuramoto equations. ArXiv (2022) doi:10.48550/arxiv.2210.00784.

For generic (but symmetric) coefficients, the system is Bernshtein/Kushnirenko-general.

A system of Laurent polynomials is Bernshtein-general if its number of isolated complex solutions is equal to the BKK bound (the mixed volume of the Newton polytopes).

Note: This is a special property of the algebraic formulation we chose. Other formulations may not be Bernshtein-general.

At most 6 zeros

$\mathcal{E}$ is 1-dimensional and is defined by a binomial system.

It contains a "torus orbit": (image of a monomial map) \[ (k_{10},k_{12},k_{32},k_{30})(t) = (k t^1, k t^1, k t^1, k t^1) \] is in $\mathcal{E}$ for almost all $t \in \mathbb{C}^*$. ($k \in \mathbb{C}^*$ are arbitrary)

This is a general phenomenon that appears in all power-flow systems on graphs with chordless even cycles.

System with "toric deficiency"

Consider a square system of $n$ Laurent polynomials \[ F(\mathbf{x}; \mathbf{c}) = F(x_1,\ldots,x_n; c_1,\ldots,c_m) \] parametrized by its coefficients $c_1,\ldots,c_m$ (one for each monomial) with supports $A = (A_1,\ldots,A_n)$ spanning $\mathbb{R}^n$.

Coefficients do not provide a crisp parametrization. We must form the quotient modulo a group action. Coefficients can still be proxy parameters.

For generic coefficients $\mathbf{c}$, the $\mathbb{C}^*$-root count is a positive constant.

(Bernshtein) This constant is $\operatorname{MV}(A_1,\ldots,A_n)$. (BKK)

We are interested in: There is a set of $E$ of parameters such that \[ \text{Isolated $\mathbb{C}^*$-zero count of } F(\mathbf{x}; \mathbf{c}) \lneq \text{BKK bound} \quad \forall \mathbf{c} \in E. \]

How many zeros? What happened to the missing ones?

Special case: Torus orbit in an exceptional stratum $\mathcal{E}$

We are interested: There is a set of $E$ of parameters such that \[ \text{Isolated zero count of } F(\mathbf{x}; \mathbf{c}) \lneq \text{BKK bound} \quad \forall \mathbf{c} \in E. \]

A 1-parameter torus orbit in $\mathcal{E}$ is the image of a map \[ t \;\mapsto\; \mathbf{c} \circ t^{\mathbf{w}} = (c_1 t^{w_1} , \ldots, c_m t^{w_m}) \in E \quad\text{for almost all } t \in \mathbb{C}^* \]

Not all such torus orbits are interesting.

An orbit $t \mapsto \mathbf{c} \circ t^{\mathbf{w}}$ in $E$ is trivial if $\mathbf{w}$ is in the row space of the Cayley embedding matrix $\hat{A}$. (...and consider other symmetries)

Systems with an exceptional stratum containing a non-trivial torus orbit are said to have toric deficiency.

Example: The all-PV/PQ system on $C_{2n}$ has toric deficiency.

Example: The all-PV/PQ system on any graph that contains a chordless even cycle has toric deficiency.

The root count for generic parameters in $E$ \[ \sum_{v} \text{Root count of } \operatorname{init}_v F(\mathbf{x}; \mathbf{c} \circ t^{\mathbf{w}}) \] for generic $\mathbf{c} \in E$.

Applying this to the power-flow equations

For the (all-PV) power-flow system derived from an even cycle $C_N$. Counting multiplicity, the complex root count is \[ \left[ N \binom{N-1}{N/2 - 1} - \beta \right], \] where $\beta$ is the number of balanced cycles. (Less than BKK bound)

Stratum-confined polyhedral homotopy

Consider the polyhedral homotopy $H : (\mathbb{C}^*)^n \times \mathbb{C}^* \to \mathbb{C}^n$: \[ H(\boldsymbol{x},t) = F(\boldsymbol{x}; \boldsymbol{c}_0 \circ t^{\boldsymbol{w}}). \]

If the image of $t \mapsto \boldsymbol{c}_0 \circ t^{\boldsymbol{w}}$ is in the closure of an exceptional stratum $\mathcal{E}$, then for generic $\boldsymbol{c}_0$ in $\mathcal{E}$, every $\mathbb{C}^*$-zero of $F(\boldsymbol{x}; c_0)$ is reached by a homotopy path defined by $H(\boldsymbol{x},t) = 0$. And the total number of paths is strictly less than the BKK bound.

Sketch of proof. For generic $\boldsymbol{c}_0 \in \mathcal{E}$ and $t$, the number of $\mathbb{C}^*$-zeros $H(\boldsymbol{x},t)$ is a constant, which is less than the BKK bound...
