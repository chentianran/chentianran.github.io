# {{ page.title }}

The following curve is defined by the equation in polar coordinate system:
<p>
\[
    r = c \cdot \theta
\]
</p>
where the constant $$c$$ determines how "tight" the spiral is.
The same curve can be represented by the parametric equation:
<p>
\[
  \begin{align*}
    r &= c \cdot t \\
    \theta &= t
  \end{align*}
\]
</p>
