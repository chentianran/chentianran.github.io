Arc-length computation

Finding the length of a straight line segment is easy, thanks to the Pythagorean Theorem.

In applications, we often need to compute the length of a curve that are not straight lines. In the context of this course, such problems are known as "arc-length' computation. This can be done using definite integrals.

For instance, if we ignore air resistance, an ball tossed at an angle will trace out a perfect parabola in the air. Can we compute the total distance the ball has traveled?

It turns out, as long as the curve can be expressed as the graph of a continuously differentiable function $y = f(x)$ (or $x = g(y)$), we can express the arc-length as a definite integral.

The length of the segment of a curve defined by a continuously differentiable function $y = f(x)$ for $a \le x \le b$ is given by \[L = \int_a^b \sqrt{ 1 + \left[ f'(x) \right]^2 } dx.\]

Here, the length of the segment of a curve is simply understood intuitively as the distance between two points along a section of a curve. Defining this concept rigorously is a much deeper problem that you will study in analysis.

Express the length of the curve defined by $y = 1-x^2$ between $x=-1$ and $x=1$.

Solution. The curve is the graph of the continuously differentiable function $y = f(x) = 1 - x^2$ between $x=-1$ and $x=1$.

We can compute that \[f'(x) = \frac{d}{dx} (1 - x^2) = -2x.\]

Therefore, the arc-length of the curve between $x=-1$ and $x=1$ is \[\int_{-1}^{1} \sqrt{1 + (-2x)^2} dx.\]

Note that here we require the function $y=f(x)$ to be not only differentiable but also continuously differentiable, i.e., its derivative is also continuous on the interval we are considering. This is important. Without this condition, this framework may fall apart.

To see an example, what do you think is the arc-length of the segment of the graph of \[ f(x) = \begin{cases} \frac{ \sin \left( \frac{1}{x^2} \right) }{x^2} &\text{if } x \ne 0 \\ 0 &\text{otherwise } \end{cases} \] between $x=-1$ and $x=1$? Note that this function is continuous but its derivative is not continuous.
