Inner product and frequency analysis

Extracing frequency information using inner product structure on waves

The analysis and generation of many forms of analog signals (e.g. sound signal) rely heavily on the inner product space structure on the set of waves. Based on our previous discussion we introduce the basic idea behind frequency information extraction.

Vector space of waves

For range of frequencies, say $k=1,\ldots,N$, we studied the vector space \[ V = \left\{ \sum_{k=1}^N a_k \cos(2k\pi t) + b_k \sin(2k\pi t) \;:\; a_k,b_k \in \mathbb{R} \right\}. \]

Combining waves

Sine (and cosine) waves of different amplitudes can be combined to generate much more complicated "waves". \[ f(t) = 1.5 \sin(3 \cdot 2\pi t) + 0.5 \cos(2 \cdot 2\pi t) + 0.2 \sin(9 \cdot 2\pi t) \]

An orthonormal basis

We have already shown that the set \[ \{ \sin( 2k \pi t), \cos( 2k \pi t) \}_{k=1}^N \] for an orthogonal basis for the vector space \[ V = \left\{ \sum_{k=1}^N a_k \cos(2k\pi t) + b_k \sin(2k\pi t) \;:\; a_k,b_k \in \mathbb{R} \right\}. \]

This basis is quite useful by itself. However, we can further improve it by "normalize" (i.e. scale) each basis element so that they are all unit vectors. The result is an orthonormal basis.

This can be done simply by "dividing" each of the basis element by their norm (i.e., their length).

Normalizing the sine function

Let $s_k(t) = \sin(2k\pi t)$, then we can calculate \[ \langle s_k, s_k \rangle = \int_{-\frac{1}{2}}^{\frac{1}{2}} \sin^2(2k \pi t) \, dt = \frac{1}{2} \]

Therefore, \[ \| s_k \| = \sqrt{ \langle s_k, s_k \rangle } = \sqrt{ \frac{1}{2} } \] This norm is related to how we scale the $t$ axis.

Then its "normalized" version \[ \sqrt{2} \sin(2k \pi t) \] is a unit vector in the inner product space $V$. This gives us a standarized version of the sine function that is independent from our choice in how the $t$ variable is measured.

Normalizing the cosine function

Similarly, let $c_k(t) = \cos(2k\pi t)$, then we can calculate \[ \langle c_k, c_k \rangle = \int_{-\frac{1}{2}}^{\frac{1}{2}} \cos^2(2k \pi t) \, dt = \frac{1}{2} \]

Therefore, its "normalized" version \[ \sqrt{2} \cos(2k \pi t) \] is also a unit vector in the inner product space $V$.

An orthonormal basis

Based on the calculations above, the set \[ \mathcal{B} = \{ \sqrt{2} \sin( 2k \pi t), \, \sqrt{2} \cos( 2k \pi t), \, \} \] form an orthonormal basis for the vector space $V$.

That is,

  • $\mathcal{B}$ is linearly independent
  • $\mathcal{B}$ spans $V$
  • Every vector in $\mathcal{B}$ is orthogonal to every other vector in $\mathcal{B}$
  • Every vector in $\mathcal{B}$ is a unit vector

These properties make an orthonormal basis particularly useful when combined with the operation of inner products.

Properties of an orthonormal basis

Example. Suppose the set $B = \{ \mathbf{b}_1, \mathbf{b}_2, \mathbf{b}_3, \mathbf{b}_4 \}$ form an orthonormal basis for a vector space $V$, and $\mathbf{v}$ is a vector in $V$ such that \[ \mathbf{v} = c_1 \mathbf{b}_1 + c_2 \mathbf{b}_2 + c_3 \mathbf{b}_3 + c_4 \mathbf{b}_4. \]

Extracting frequency information

Going back to the "inverse" problem...

We want to figure out the coefficients for the sine and cosine functions of different frequencies that will recreate this wave as a linear combination.

Extracting frequency information

Given any $f \in V$, it must be a linear combination of $\mathcal{B}$ since $\mathcal{B}$ form a basis of $V$.

That is, \[ f(t) = \sum_{k=1}^N a_k \sqrt{2} \cos(2k\pi t) + b_k \sqrt{2}\sin(2k\pi t). \] for some real numbers $a_1,b_1,a_2,b_2,\ldots,a_N,b_N$.

Now suppose we only have the function (wave) $f(t)$ itself, and the coefficients $\{ a_k, b_k \}$ are lost. Thanks to the orthogonal basis $\mathcal{B}$, we can recover the coefficients.
