Advanced data structures

What is a cell array?

A cell array is a data type with indexed data containers called "cells" where each cell can contain any type of data.

  • ...allow mixing data types
  • ...its like an array of containers

Creation

Use { } operator to create cell arrays: 


        C = { 1, 2, 3, 'Hello', {11;22;33} };

Use { } for empty cell arrays: 


        C = {};

The "cell" function can create array of any size.

Accessing cell array

You can access cells in a cell array using two operators.

  • "( )" refer to sets of cells
  • "{ }" refer to content of the cells

Syntax for indexing is the same as that for arrays.

What is a structure array

A structure array is a data type that groups related data using data containers called fields.

  • Fields have names
  • Each field can contain data of any type
  • We use "." notation for fields

        s.width  = 100;
        s.height = 200;

Compare these concepts: Arrays, cell arrays, Structures.

Example


    data.x = linspace(0,2*pi);
    data.y = sin(data.x);
    data.title = 'y = sin(x)'

Structured array is a nice to group related data together.

The structure function can be used to create structures of any size.
