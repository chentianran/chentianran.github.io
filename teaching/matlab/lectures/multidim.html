Multidimensional arrays

Arrays

In MATLAB, arrays are classified according to their "dimensions".

  • A vector is a 1-dimensional array.
  • A matrix is a 2-dimensional array.
  • A multidimensional array is an array with more than two dimensions.
  • Dimension can also be changed dynamically.

3D arrays

A 3-dimensional (3D) array consists of several "pages". Each page is a 2D array (matrix).

An entry in a 3D array requires 3 subscripts to be located.

We usually create a 2D array first, and then extend it by adding pages. 


        A = [ 1 2 ; 3 4 ]
        A(:,:,2) = [ 5 6 ; 7 8 ]

Other ways

We can also create 3D arrays (or multidimensional arrays in general) using the these functions:

  • cat
  • reshape
  • repmat
  • meshgrid

Shape and size

We often need to get the dimension and size of a 3D array.

  • size
  • ndims
  • numel

Array indexing

  • A(3,:)
  • A(:,3)
  • A(1:3,2)
