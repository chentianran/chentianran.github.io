Laplace transform

Named after Pierre-Simon Laplace, the Laplace transform is an "integral transform" that transforms a function of time to a function of complex frequency. It is an important tool in mathematics, physics, and engineering.

In the context of ordinary differential equations, it is an extremely useful tool for solving initial value problems.

For example, difficult initial value problems like \[ y'' + 4y = e^{-3t} \sin(2t), \quad y(0) = 0, \quad y'(0) = 1 \] can be solved relatively easily through Laplace transform.

Review: Improper integrals

In order to make sense of the following discussion, a clear understanding of improper integrals of the form $\int_0^\infty f(t) dt$ is necessary.

It is defined to be \[ \int_0^\infty f(t) dt = \lim_{h \to \infty} \int_0^h f(t) dt \] if this limit exists (and is finite). In this case, we say the improper integral converges. Otherwise, we say it diverges.

Laplace transform

For a function $f(t)$ that is defined for all $t \ge 0$, its Laplace transform is the function $F(s)$ in a complex variable $s$ given by \[ F(s) = \mathcal{L}\{f(t)\} = \int_0^\infty e^{-s t} \, f(t) \, dt \] if this improper integral converges.

In this definition, we allow $s$ to be complex even when starting with real functions, and $s$ is often known as the complex frequency parameter.

Example. Let's compute the Laplace transform of $f(t) = 1$:

$\displaystyle \mathcal{L}\{ 1 \} = \int_0^\infty e^{-st} 1 \, dt$ $\displaystyle = \lim_{x \to \infty} \int_0^x e^{-st} 1 \, dt$ $\displaystyle = \lim_{x \to \infty} -\frac{e^{-sx}}{s} + \frac{1}{s}$ $\displaystyle = \frac{1}{s}$

which is valid for $s > 0$. Since we allow $s$ to be complex, this result is actually valid for any $s$ with a positive real part.

Another example

Let's compute the Laplace transform of the function $f(t) = e^{at}$ for a nonzero constant $a$.

$\displaystyle \mathcal{L}\{ e^{at} \} = \lim_{x \to \infty} \int_0^x e^{-st} e^{at} \, dt$ $\displaystyle = \lim_{x \to \infty} \int_0^x e^{(a-s)t} dt$ $\displaystyle = \lim_{x \to \infty} \left[ \frac{e^{(a-s)t}}{a-s} \right]_0^x$
$\displaystyle = \lim_{x \to \infty} \frac{e^{(a-s)x}}{a-s} - \frac{1}{a-s}$ $\displaystyle = \frac{1}{s-a}$

which is valid if the real part of $s-a$ is positive.

Therefore, we can conclude that \[ \mathcal{L}\{ e^{at} \} = \frac{1}{s-a} \] which is defined for any $s$ for which the real part of $s-a$ is positive.

The $\mathcal{L}$ operator

In this definition, \[ \mathcal{L}\{f(t)\} = \int_0^\infty e^{-s t} \, f(t) \, dt, \]

$\mathcal{L}$ is considered as an operator in the sense that it takes in a function and returns another function as output. Notice that the input and output functions do not have the same variable. That is, it also transform the $t$-domain (time domain) to the $s$-domain (complex frequency domain).

Caveat: a Laplace transform may not exist

Not all functions have Laplace transforms. Indeed, for the Laplace transform $\mathcal{L}\{ f(t) \}to make sense, we require $f(t)$ to be...

  • continuous for $t > 0$ except at a finite number of jump discontinuities;
  • of at most "exponential order", i.e., $|f(t)|$ can be bounded by $M e^{ct}$ for some sufficiently large real number $M$;

Intuitively speaking, functions that have "too many" discontinuities, "strange" discontinuities, or grow too fast, may not have Laplace transforms.

Laplace transform is a linear operator

It is easy to verify that $\mathcal{L}\{ 0 \} = 0$. That is, the Laplace transform of the zero function (in $t$) is the zero function (in $s$).

Moreover, for functions $f_1(t)$ and $f_2(t)$ whose Laplace transforms are defined and (complex) constants $c_1,c_2$, \[ \mathcal{L} \{ c_1 f_1 + c_2 f_2 \} = c_1 \, \mathcal{L} \{ f_1 \} + c_2 \, \mathcal{L} \{ f_2 \}. \]

This property is called linearity, and we say $\mathcal{L}$ is a linear operator.

Laplace transform is almost invertible

The hope is, knowing the Laplace transform $F(s)$ of a function $f(t)$, we can recover $f(t)$. This is the key in solving certain differential equations.

It turns out $\mathcal{L}$ is not one-to-one, i.e., multiple functions can produce the same Laplace transform. Inverting $\mathcal{L}$ is thus not possible in general.

However, if restricted to continuous functions, $\mathcal{L}$ is indeed one-to-one. I.e., there is a unique continuous function $f(t)$ for which $F(s) = \mathcal{L} \{ f(t) \}$.

Under this assumption, we write \[ f(t) = \mathcal{L}^{-1} \{ F(s) \} \] and call this the inverse Laplace transform.

Laplace transform of derivatives

For solving differential equations, the most important property of Laplace transform is how it interact with derivatives.

If the involved Laplace transforms are defined, \begin{align*} \mathcal{L} \{ y'\,\, \} &= s \mathcal{L} \{ y \} - y(0) , \\ \mathcal{L} \{ y'' \, \} &= s^2 \mathcal{L} \{ y \} - sy (0) - y'(0) ,\\ \mathcal{L} \{ y''' \} &= s^3 \mathcal{L} \{ y \} - s^2 y (0) - s y' (0) - y''(0) \\ &\vdots \\ \mathcal{L} \left\{ y^{(n)} \right\} &= s^n \mathcal{L} \{ y \} - \sum_{k=1}^n s^{n-k} y^{(k-1)}(0). \end{align*}

An important observation: $\mathcal{L}$ turns derivatives of $y$ into algebraic expressions in $\mathcal{L}\{y\}$, $s$, and the initial values.

Solving differential equations: an example

For a differential equation in $y(t)$, we can apply $\mathcal{L}$ to the entire equation, which will transform the differential equation into a new algebraic equation involving $Y(s)$ and $s$.

We can then solve the $Y(s)$ and then apply the inverse Laplace transform $\mathcal{L}^{-1}$ to find the original function $y$. This is the main idea.

A simple example

Problem. Find a solution $y(t)$ to the DE \[ y'' = -y + e^{-2t} \quad \text{with } \quad y(0) = y'(0) = 0. \]

Solution. Applying Laplace transform, we get \[ \mathcal{L} \{ y'' \} = -\mathcal{L} \{y\} + \mathcal{L} \{ e^{-2t}\}. \]

Using the table above, this turns into \[ s^2 Y - sy(0) - y'(0) = -Y + \frac{1}{s + 2}. \]

Then using the initial conditions, this simplifies to \[ s^2 Y = -Y + \frac{1}{s + 2} \quad\text{i.e.,}\quad Y = \frac{1}{(s^2+1)(s+2)} \] Can you recognize what $y = \mathcal{L}^{-1} \{ Y \}$ is?

What is Laplace transform?

Recall that given coefficients function $c_k = c(k)$, a power series can be formed as \[ \sum_{k=0}^\infty c_k \, x^k \;=\; \sum_{k=0}^\infty x^k \, c(k). \]

A continuous version of this construction will be \[ \int_0^\infty x^t \, c(t) \, dt \;=\; \int_0^\infty e^{(\ln x) t} \, c(t) \, dt. \] Clearly, we need $x < 1$, i.e., $\ln x < 0$ for this integral to have any hope of converging. With that in mind, we write $s = -\ln x$, and this produces the Laplace transform $\mathcal{L}\{c(t)\} = \int_0^\infty e^{-st} \, c(t) \, dt.$ Therefore, the Laplace transform can be viewed as a continuous version of a power series construction.
